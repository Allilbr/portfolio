​ДИЗАЙН И ЭСТЕТИКА (UI/UX)
​Интерфейс выполнен в премиальной, драматичной и глубокой цветовой палитре, которая подчеркивает дерзинку текста.
​Основной фон сайта выполнен в теплом кремовом цвете (слоновая кость), что обеспечивает мягкое чтение текста. Акцентным цветом для важных секций, цитат и выделения ключевых фраз (вроде «ненавижу химию») выбран глубокий бордовый оттенок(любимый цвет). Для главных кнопок, рамок аватаров и декоративных линий используется благородное приглушенное золото, а основной текст написан контрастным угольно-черным цветом.
​В типографике используется элегантное сочетание шрифтов с засечками в заголовках для придания литературного шика и современных простых шрифтов для легко читаемого текста. Интерактивность включает в себя удобные окна для карточек вдохновения, эффекты при наведении на кнопки и плавный скролл по меню.
​ТЕХНОЛОГИЧЕСКИЙ СТЕК
​Проект реализован с использованием классических фронтенд-технологий:
​HTML для семантической верстки, обеспечивающей доступность и правильную структуру документа.
​CSSдля создания адаптивного дизайна под смартфоны и десктопы, кастомных круглых рамок с золотыми границами и плавных анимаций.
​JavaScript для реализации логики открытия и закрытия окон (например, с описанием качеств Скарлетт О’Хара) и плавной навигации.
​GitHub Pages для быстрого и стабильного хостинга сайта напрямую из репозитория.
​СТРУКТУРА РЕПОЗИТОРИЯ
​В корне проекта находится файл index.html (главная страница сайта с разметкой). В папке css лежит файл style.css со всеми стилями, цветами и адаптивной сеткой. В папке js находится файл main.js, отвечающий за интерактив. Папка img содержит оптимизированные изображения: аватар для главного экрана, фото для истории и скриншоты выполненных проектов. Замыкает структуру файл README.md с описанием проекта.
